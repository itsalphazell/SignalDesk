## Signal Desk

Signal Desk is a premium B2B interactive prototype built to show product-grade interface systems across overview, analytics, onboarding, projects, and settings.

## Positioning

- Interactive prototype repo for portfolio use
- Measured premium visual system rather than marketing-page styling
- Built to prove dense information design, calmer analytics, and richer client-side product behavior

## Public Links

- Live prototype: `https://signaldesk.pages.dev`
- Repository: `https://github.com/itsalphazell/SignalDesk`

## Commands

```bash
npm run dev
npm run build
```

- `/`
- `/projects`

## Notes

- This repo is intended as a public interactive prototype case study in a wider portfolio
- Cloudflare Pages is connected to the GitHub repo, so pushes to `main` trigger production deploys automatically
