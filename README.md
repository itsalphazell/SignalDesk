## Signal Desk

Signal Desk is a premium B2B dashboard concept built to show product-grade front-end UI: overview, analytics, onboarding, projects, and settings.

## Positioning

- Static concept repo for portfolio use
- Measured premium visual system rather than marketing-page styling
- Built to prove dense information design and responsive product UI

## Public Links

- Live concept: `https://signaldesk.pages.dev`
- Repository: `https://github.com/itsalphazell/SignalDesk`

## Commands

```bash
npm run dev
npm run build
```

## Routes

- `/`
- `/projects`
- `/analytics`
- `/onboarding`
- `/settings`

## Notes

- This repo is intended as a public concept case study in a wider portfolio
- Cloudflare Pages is connected to the GitHub repo, so pushes to `main` trigger production deploys automatically
